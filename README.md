# Гороскоп (Horoscope App)

Добро пожаловать в репозиторий приложения **Гороскоп**. Это современное и удобное веб-приложение для получения ежедневных гороскопов и астрологических прогнозов. 

---

## 🚀 Как запустить проект локально

Убедитесь, что у вас установлен **Node.js** (версии 16 и выше).

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/AndreyViht/goroscop.git
   ```

2. **Перейдите в папку проекта:**
   ```bash
   cd goroscop
   ```

3. **Установите зависимости:**
   Для установки необходимых библиотек выполните команду:
   ```bash
   npm install
   ```

4. **Запустите сервер разработки:**
   ```bash
   npm run dev
   ```
   После этого приложение будет доступно в вашем браузере по адресу `http://localhost:5173`.

---

## 🛠 Сборка для продакшена (Deploy)

Если вы хотите собрать оптимизированную версию приложения для загрузки на хостинг, выполните:
```bash
npm run build
```
Готовые статические файлы появятся в папке `dist/`.

Для максимально комфортной работы мы рекомендуем использовать **Visual Studio Code** со следующими плагинами:

В проекте уже присутствуют конфигурации `.eslintrc` и `Prettier`, поэтому ваш код будет форматироваться автоматически при сохранении.
